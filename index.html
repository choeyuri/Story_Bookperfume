북퍼퓸, 향으로 이야기를 그려나가다
이상한 나라의 앨리스 북퍼퓸
순수한 모험과 싱그러움을 느낄수있는 플로랄 향기. 이상한 나라를 방황하는 앨리스처럼 신비로운 모험을 겪게 해주는 향을 느껴보세요.
Top : Bergamot / Orange Blossom / Mandarin
Middle : Mimosa / Jasmin / Rose
Base : Patchouli / Tonka bean / Vetiver
빨간머리 앤 북퍼퓸
매일매일 즐거움이 가득한 빨간머리 앤처럼 발랄하고 사랑스러운 프루티 프래그런스. 일상에 지쳐있는 당신을 위한 사랑스러운 향기입니다.
Top : Cirtus / Peacch / Apple
Middle : Lily / Jasmin / Rose
Base : Powdery / Musk / Iris
노인과 바다 북퍼퓸
바다의 청량한 향기와 더불어 노인의 깊고 중후한 멋이 담겨있는 워터릴리 바탕의 프래그런스. 푸른 바다의 반짝임을 떠오르게 하는 상쾌한 향기를 느낄 수 있습니다.
Top : Citrus / Waterlily / Grapefruit
Middle : Floral / Magnolia / Jasmine
Base : Woody / Cardamom / Veriver
셜록홈즈 북퍼퓸
셜록홈즈의 다양한 이야기를 표현하여 여름의 런던거리를 연상시키는 시원하고 달콤한 향기. 다양한 등장인물의 매력을 담은 풍성한 향을 즐겨보세요.
Top : Green / Citrus / Bergamot
Middle : Fruity / Water / Apple
Base : Woody / Moss / Sandalwood
로미오와 줄리엣 북퍼퓸
로미오와 줄리엣의 짧고 강렬한, 비극적인 사랑을 닮은 플로랄 그린머스크 베이스퍼퓸. 달콤한 사랑과 무거운 비극을 한번에 담긴 입체적인 향기를 담았습니다.
Top: Daisy / Rose / Melon
Middle : Damaskrose / Jasmine / Violet
Base : Agarwood / Praline / Musk
